 Radix Sort
  Space: O(n + d) // n: num elements, d: current radix
  Time: O(n*k) // n: num element, k: num digits in largest number | ceil(log_d(biggest number))
 Merge Sort
  Space: O(n) // n: num elements
  Time: O(n*logn) // n: num elements
 Bucket/Counting Sort
  Space: O(r) // r: biggest - smallest
  Time: O(r+n) // r: range, n: range
 Quick Sort
  Space: O(logn) - worst: O(n)
  Time: O(n*logn) - worst: O(n^2);
